# Caderno-Tem-tico---An-lise-T-cnica
Este projeto foi desenvolvido utilizando o NotebookLM como ferramenta de estudo e organização do conhecimento.  O tema escolhido foi Análise Técnica, com foco na interpretação de gráficos, tendências, ciclos de mercado, metodologia ICT, Teoria de Elliott e gerenciamento de risco.
