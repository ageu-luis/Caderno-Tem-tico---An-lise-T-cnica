# Caderno Temático - Análise Técnica
Este projeto foi desenvolvido utilizando o NotebookLM como ferramenta de estudo e organização do conhecimento.  O tema escolhido foi Análise Técnica, com foco na interpretação de gráficos, tendências, ciclos de mercado, metodologia ICT, Teoria de Elliott e gerenciamento de risco.

Objetivos
Compreender os fundamentos da análise técnica.
Estudar o comportamento institucional.
Entender ciclos de mercado.
Aprender a identificar suportes e resistências.
Desenvolver pensamento operacional baseado em probabilidade.

Fontes Utilizadas:
Foram utilizadas 25 fontes entre,
Livros de análise técnica,
Apostilas,
Vídeos educacionais,
Materiais sobre ICT,
Materiais sobre Ondas de Elliott,
Estratégias operacionais.

Engenharia de Prompts

Prompt:
Explique o que é análise técnica de forma resumida.
Resultado:
A análise técnica busca identificar tendências através do estudo dos preços, volume e comportamento coletivo dos participantes do mercado.

Prompt:
Qual a relação entre ciclos de mercado e ciclos econômicos?
Resultado:
O mercado normalmente antecipa movimentos econômicos futuros, refletindo expectativas antes da divulgação dos dados oficiais.

Prompt: 
Explique a diferença entre suporte e resistência utilizando exemplos práticos.
Resultado:
Suporte representa uma região de interesse comprador.
Resistência representa uma região de interesse vendedor.
Foi observado também o conceito de bipolaridade, onde suportes rompidos tornam-se resistências e vice-versa.

Dificuldades Encontradas:
Respostas muito genéricas.
Necessidade de refinar prompts.
Comparação entre diferentes autores.
Validação das respostas utilizando múltiplas fontes.

Principais Aprendizados: 
O mercado antecipa a economia.
Volume confirma movimentos.
Nenhum indicador funciona isoladamente.
Gestão de risco é mais importante que prever o mercado.
Liquidez institucional influencia os movimentos de preço.
Suportes e resistências são zonas e não linhas exatas.

Glossário:
Termo	Definição,
Suporte	Região de interesse comprador,
Resistência	Região de interesse vendedor,
RSI	Índice de Força Relativa,
ICT	Inner Circle Trader,
Elliott	Teoria baseada em ciclos de ondas,
FVG	Fair Value Gap.
